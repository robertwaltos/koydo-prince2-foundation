# koydo-prince2-foundation

PRINCE2 Foundation exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM158 |
| Name | PRINCE2 Foundation |
| Country | Global |
| Category | Professional certification |
| Tier | Tier 6 |
| Wave | Wave 8 |
